Bindable Functions

Split Window Manipulation:

window_split_horz

default bindings:
  Emacs : 'C-x 3'
  Vi    : ''
  Other : ''
Split the current focused screen horizontally into two screen.

window_split_vert

default bindings:
  Emacs : 'C-x 2'
  Vi    : ''
  Other : ''
Split the current focused screen vertically into two screen.

window_delete

default bindings:
  Emacs : 'C-x 0'
  Vi    : ''
  Other : ''
Remove the current focused screen (the buffer shown in the screen will be un-touched).

window_delete_other

default bindings:
  Emacs : 'C-x 1'
  Vi    : ''
  Other : ''
Remove all screens but the current focused one (the buffer shown in those screens will be un-touched).

Buffer Manipulation:

buffer_open

default bindings:
  Emacs : 'C-x C-f'
  Vi    : ''
  Other : 'C-o'
Switch the current focused screen to the browser to open a new file (buffer).

buffer_save

default bindings:
  Emacs : 'C-x C-s'
  Vi    : ''
  Other : 'C-s'
Save the buffer shown in the current focused screen.

buffer_save_as

default bindings:
  Emacs : 'C-x C-w'
  Vi    : ''
  Other : 'C-S'
Save the buffer shown in the current focused screen as another (possibly the same) file.

buffer_close

default bindings:
  Emacs : 'C-x k'
  Vi    : ''
  Other : 'F4'
Close the buffer shown in the current focused screen. This will also cause all screens which have the same buffer shown to switch to other buffers.

buffer_undo

default bindings:
  Emacs : 'C-x u'
  Vi    : ''
  Other : 'C-z'
Undo the last change mado to the buffer in the current focused screen. This will also cause all screens which have the same buffer to revert.

buffer_redo

default bindings:
  Emacs : ''
  Vi    : ''
  Other : 'C-y'
Redo the last "Undo" mado to the buffer in the current focused screen. This will also cause all screens which have the same buffer to change.

Tab Manipulation:

tabbar_mode

default bindings:
  Emacs : 'F7'
  Vi    : ''
  Other : 'F7'
Switch between "on", "grouped", and "off" mode.

tabbar_prev

default bindings:
  Emacs : 'C-Left'
  Vi    : ''
  Other : 'C-Left'
Switch the current focused screen to show the content of the previous tab's buffer. This will turn on tabbar automatically.

tabbar_next

default bindings:
  Emacs : 'C-Right'
  Vi    : ''
  Other : 'C-Right'
Switch the current focused screen to show the content of the next tab's buffer. This will turn on tabbar automatically.

Editor Related Functions:

complete

default bindings:
  Emacs : 'Tab'
  Vi    : ''
  Other : 'Tab'
Complete the current typing.

complete_list

default bindings:
  Emacs : 'M-/'
  Vi    : ''
  Other : ''
Show completion list of the current typing; whether set of not, two successive fail in "complete" will cause the list to show.

backward_char

default bindings:
  Emacs : 'C-b'
  Vi    : ''
  Other : ''
Move the cursor to the previous character.

backward_delete_char

default bindings:
  Emacs : 'BackSpace'
  Vi    : ''
  Other : 'BackSpace'
Delete the previous character.

forward_char

default bindings:
  Emacs : 'C-f'
  Vi    : ''
  Other : ''
Move the cursor to the next character.

forward_delete_char

default bindings:
  Emacs : 'Delete'
  Vi    : ''
  Other : 'Delete'
Delete the next character.

backward_word

default bindings:
  Emacs : 'M-b'
  Vi    : ''
  Other : ''
Move the cursor to the start of the current word, or the previous word if not currently inside one.

backward_delete_word

default bindings:
  Emacs : 'M-D'
  Vi    : ''
  Other : ''
Kill to the start of the current word, or kill the previous word if not currently inside one. This will put the killed word(s) into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

forward_word

default bindings:
  Emacs : 'M-f'
  Vi    : ''
  Other : ''
Move the cursor to the end of the current word, or the next word if not currently inside one.

forward_delete_word

default bindings:
  Emacs : 'M-d'
  Vi    : ''
  Other : ''
Kill to the end of the current word, or kill the next word if not currently inside one. This will put the killed word(s) into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

backward_line

default bindings:
  Emacs : 'C-a'
  Vi    : ''
  Other : ''
Move the cursor to the start of the current line.

backward_delete_line

default bindings:
  Emacs : 'C-K'
  Vi    : ''
  Other : ''
Kill to the start of the current line. This will put the killed line(s) into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

forward_line

default bindings:
  Emacs : 'C-e'
  Vi    : ''
  Other : ''
Move the cursor to the end of the current line.

forward_delete_line

default bindings:
  Emacs : 'C-k'
  Vi    : ''
  Other : ''
Kill to the end of the current line, or kill the entire current line if at line start, or kill the new-line character if at line end. This will put the killed line(s) into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

backward_para

default bindings:
  Emacs : 'M-{'
  Vi    : ''
  Other : ''
Move the cursor to the start of the current paragraph, or the previous paragraph if not currently inside one or at the start of a paragraph.

backward_delete_para

default bindings:
  Emacs : 'M-K'
  Vi    : ''
  Other : ''
Kill to the start of the current paragraph, or kill the previous paragraph if not currently inside one or at the start of a paragraph. This will put the killed paragraph(s) into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

forward_para

default bindings:
  Emacs : 'M-}'
  Vi    : ''
  Other : ''
Move the cursor to the end of the current paragraph, or the next paragraph if not currently inside one or at the end of a paragraph.

forward_delete_para

default bindings:
  Emacs : 'M-k'
  Vi    : ''
  Other : ''
Kill to the end of the current paragraph, or kill the next paragraph if not currently inside one or at the end of a paragraph. This will put the killed paragraph(s) into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

kill_region

default bindings:
  Emacs : 'C-w'
  Vi    : ''
  Other : 'C-x'
Kill the selected region. This will put the killed region into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

kill_ring_save

default bindings:
  Emacs : 'M-w'
  Vi    : ''
  Other : 'C-c'
Save the selected region into the kill-ring and unset the selection mark. This will put the saved region into the kill-ring, which can then be yanked back. (see "kill_ring_yank" and "kill_ring_yank_pop" for more details.)

kill_ring_yank

default bindings:
  Emacs : 'C-y'
  Vi    : ''
  Other : 'C-v'
Restore the last text (region) being yanked (if any) or being killed.

kill_ring_yank_pop

default bindings:
  Emacs : 'M-y'
  Vi    : ''
  Other : ''
Replace the just-yanked text (region) with the one before it; require "kill_ring_yank" as an immidiate precedent command.

set_mark_command

default bindings:
  Emacs : 'C-@'
  Vi    : ''
  Other : ''
Set the selection mark at the current cursor position.

set_mark_move_left

default bindings:
  Emacs : 'S-Left'
  Vi    : ''
  Other : 'S-Left'
Set the selection mark on the initial invoke, move the cursor 1 character left on each initialized invokes.

set_mark_move_right

default bindings:
  Emacs : 'S-Right'
  Vi    : ''
  Other : 'S-Right'
Set the selection mark on the initial invoke, move the cursor 1 character right on each initialized invokes.

set_mark_move_up

default bindings:
  Emacs : 'S-Up'
  Vi    : ''
  Other : 'S-Up'
Set the selection mark on the initial invoke, move the cursor 1 char up (if not first line) on each initialized invokes.

set_mark_move_down

default bindings:
  Emacs : 'S-Down'
  Vi    : ''
  Other : 'S-Down'
Set the selection mark on the initial invoke, move the cursor 1 char down (if not last line) on each initialized invokes.

set_mark_move_start

default bindings:
  Emacs : 'S-Home
  Vi    : ''
  Other : 'S-Home
Set the selection mark on the initial invoke, move the cursor to the start of the buffer on each initialized invokes.

set_mark_move_end

default bindings:
  Emacs : 'S-End
  Vi    : ''
  Other : 'S-End
Set the selection mark on the initial invoke, move the cursor to the end of the buffer on each initialized invokes.

Top-Level Functions

  Vi    : ''
  Other : 'C-p'
Show the Config Console.

prog_show_error

default bindings:
  Emacs : 'C-c e'
  Vi    : ''
  Other : 'C-J'
Show the Error Console.

prog_show_about

default bindings:
  Emacs : ''
  Vi    : ''
  Other : ''
Show the About dialog.

prog_quit

default bindings:
  Emacs : 'C-x C-c'
  Vi    : ''
  Other : 'C-q'
Quit the program. Any modified buffer except *scratch* will be prompt for saving.

zPE Related Functions:

zPE_submit

default bindings:
  Emacs : 'F9'
  Vi    : ''
  Other : 'F9'
Submit the file corresponding to the buffer shown in the current focused screen. Any unsaved change will not take effects.

zPE_submit_with_JCL

default bindings:
  Emacs : 'F8'
  Vi    : ''
  Other : 'F8'
Submit the file corresponding to the buffer shown in the current focused screen, with default JCL wrapped around. This should only be used to quick-test a program.
